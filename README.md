# 智能水利知识库管理系统

## 项目简介

智能水利知识库管理系统是基于知识图谱和向量数据库的专业水利工程知识管理平台。系统结合了先进的自然语言处理技术和知识图谱技术，为水利大坝安全监测、隐患处理、智能问答等应用场景提供全面的技术支持。

## 核心功能

### 🧠 智能问答系统
- 支持多轮对话和上下文理解
- 基于专业领域知识的精准回答
- 多种大语言模型支持

### 🕸️ 知识图谱可视化
- 直观的知识关联展示
- 交互式图谱探索
- 实时知识更新

### 📚 知识库管理
- 多格式文档支持 (PDF、Word、图片等)
- 智能文档解析和索引
- 高效的向量检索

### 📊 数据分析看板
- 系统运行状态监控
- 知识使用统计分析
- 用户行为分析

## 技术架构

- **前端**: Vue.js 3 + Ant Design Vue + TypeScript
- **后端**: FastAPI + Python 3.11+
- **数据库**: Milvus/Chroma (向量数据库) + Neo4j (图数据库)
- **文档解析**: LightRAG + MinerU + PP-Structure-V3
- **容器化**: Docker Compose

## 快速开始

### 环境要求

- Docker 和 Docker Compose
- Python 3.11+
- Node.js 18+
- pnpm 包管理器

### 安装部署

1. 克隆项目到本地
2. 配置环境变量文件
3. 启动服务

```bash
# 复制环境配置文件
cp .env.example .env

# 启动所有服务
docker-compose up -d
```

### 系统配置

首次访问需要创建管理员账户，然后可以：
- 上传水利相关文档
- 配置AI模型参数
- 设置用户权限

## 系统特色

### 专业领域适配
- 针对水利大坝安全场景深度优化
- 包含大量专业术语和知识库
- 支持隐患识别和处理建议

### 三级权限管理
- 管理员：完全控制权限

- 向量数据库支持快速语义搜索
- 知识图谱提供深层关联分析

## 使用指南

### 文档上传
支持批量上传多种格式的文档，系统会自动进行：
- 文本提取和清洗
- 实体识别和关系抽取
- 向量化和索引构建

### 智能问答
1. 在对话界面输入问题
2. 系统自动检索相关知识
3. 结合知识图谱生成答案
4. 支持追问和澄清

### 知识图谱探索
- 可视化浏览知识节点和关系
- 支持节点详情查看
- 实时路径分析
